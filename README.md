# Student's POV Tutoring Website

Student's POV is a personal tutoring and engineering portfolio website. The project is being built with HTML, CSS, and JavaScript while documenting the learning process step by step.

## Project Purpose

The site has two main goals:

- Provide step-by-step tutoring resources for engineering, math, physics, chemistry, and circuits.
- Present an employer-ready engineering portfolio with projects, resume links, and technical work.

## Current Pages

- `index.html` - Main tutoring website landing page.
- `Webpages/portfolio.html` - Employer-focused portfolio page.
- `Webpages/about.html` - About page, planned for future cleanup.
- `Webpages/intro_circuits.html` - Introductory circuits lesson page.

## Current Focus

The current focus is making the portfolio page employer-ready. This includes:

- A stronger portfolio hero section.
- A resume-focused first impression.
- Featured engineering project cards.
- Space for future project images, YouTube thumbnails, and resume links.
