ChargeLab Web

ChargeLab is an interactive Coulomb-force simulator written in C++ and compiled to WebAssembly. Add, select, drag, edit, and fix charged particles to explore electric forces.

How to Use

Select a mode, then interact with the canvas.

  • Select/Drag: click and drag particles
  • Add +: click canvas to add a positive charge
  • Add -: click canvas to add a negative charge
  • Fix Selected: keep a selected particle in place
  • Apply Changes: update the selected particle's values
Input Units & Ranges
  • Charge: −100 to 100 µC
  • Mass: 0.001 to 1 kg
  • Radius: 0.000001 to 0.0001 m (1–100 µm)
  • Position X: 0 to 5.6 m
  • Position Y: 0 to 2.8 m
  • Velocity: −10 to 10 m/s

Physical radii are enlarged on screen so particles remain visible.
